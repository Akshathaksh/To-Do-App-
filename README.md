# 📚 Student To-Do App

A modern and interactive **Student To-Do Application** built using **HTML, CSS, and JavaScript** to help students manage their daily tasks efficiently.

---

## 🚀 Features

* ✅ Add, delete, and edit tasks
* 📅 Due date management
* ⭐ Priority levels (High, Medium, Low)
* 🔍 Search tasks
* 📊 Task counter & progress tracking
* 🌙 Dark mode
* 📌 Separate sections (Pending & Completed)
* 🔔 Smart reminders for due tasks
* 🎯 Productivity streak system
* 🔊 Voice input for adding tasks
* 📥 Export tasks as file
* 🎨 Modern UI with Glassmorphism design

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage

## 💡 How to Run

1. Download or clone the repository
2. Open `index.html` in any browser
3. Start managing your tasks 🎉
